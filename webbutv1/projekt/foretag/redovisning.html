Cart systemet

Cart systemet ska vara ett system för att förvara vilka varor en andvendare vill köpa. Det ska vara möjligt att lägga till, tabort, och rensa den. Den ska förvara datan i web storage.

först tänkte jag att structuren för att förvara skulle vara så här {
     cartItemId: {
         id: aodnsadsad,
         productId: x,
         productCount: x,
     }
}

Men ångrade mig och ladde istället alla cartItems i en list istället för ett JSON object. Gjorde det senare då vissa saker som att kolla ifall en produt redan finns i carten skulle kunna bli finare implementerat.
